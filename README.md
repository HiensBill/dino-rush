# Dino Rush：恐龙蓄力跳跃小游戏

一个改造自 Chrome 小恐龙的轻量级网页跑酷游戏：  
按住蓄力、松手起跳，配合 Buff 抽卡和在线排行榜，看你能跑多远。

> 在线试玩：**https://dino-game.liulj.com/**

---

## 🎮 游戏玩法

- 按住空格 / 鼠标左键 / 手指 **蓄力**，松手后恐龙起跳  
  - 蓄力越久，跳得越高、越远  
- 躲避地面的仙人掌、坑洞和飞鸟等障碍物，**跑得越远分数越高**
- 跑到一定分数会出现 **3 张 Buff 卡牌**，从中选 1 张本局生效

当前 Buff 示例：

- 🟢 **护盾**：下一次撞到障碍物不死亡  
- 🔵 **三段跳**：在空中可以多跳一次  
- 🟣 **缩小**：恐龙体型变小，更容易钻缝隙  
- 🟡 **变轻**：重力减小，滞空时间更长  
- 🔴 **贪婪**：一段时间内得分翻倍

---

## ✨ 目前特性

- 蓄力条 + 挤压动画，手感接近主机平台跑酷
- 多种障碍物组合（地面、空中、坑洞、高速怪等），难度随分数递增
- 不同主题场景 / 配色，让跑酷过程没那么单调
- Buff 图标显示在左上角，方便随时查看当前加成
- **在线排行榜**：登录后可以上传最高分，在榜单里看到自己的昵称
- 同时支持 PC 浏览器和手机浏览器（横屏体验更佳）
