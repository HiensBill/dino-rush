# Dino Rush：恐龙蓄力跳跃游戏

一个基于 Chrome 小恐龙的 **蓄力跳跃 + Buff 抽卡 + 在线排行榜** 的进化版小游戏。  
前端部署在 **阿里云 OSS**，排行榜数据通过 **阿里云函数计算（FC） + MySQL** 持久化。

在线地址（示例）：`https://dino-game.liulj.com/`

---

## 🎮 游戏玩法简介

- 按住屏幕 / 鼠标左键 **蓄力**，松手后恐龙会按蓄力条的长度起跳  
- 蓄力条越长，跳得越高、越远  
- 撞到障碍物会游戏结束（有护盾时会抵消一次）

### Buff 抽卡系统

每到一定得分，会弹出三张随机 Buff 卡牌，从中选择一张：

- 🟢 **护盾**：抵挡一次伤害  
- 🔵 **三段跳**：在空中可以额外再跳两次  
- 🟡 **变轻**：重力减小，起跳更轻盈  
- 🟣 **缩小**：恐龙体积变小，更容易穿过缝隙  
- 🔴 **贪婪**：一段时间内分数翻倍

启用中的 Buff 会以小图标的形式显示在左上角状态栏。

## ✨ 主要特性

- ⏱️ **蓄力跳跃手感**：按住-松手的物理反馈，跳跃高度连续可控  
- 🃏 **随机抽卡升级**：从 3 张 Buff 卡片中选择一张，构建自己的强化路线  
- 🛡️ **可视化 Buff 状态栏**：当前 Buff 一目了然  
- 🏆 **全服排行榜**：后端存储 TOP 10 成绩，并在游戏内弹窗展示  
- 📱 **PC / 手机 双端适配**：点击 / 触摸均可操作  
- 🎨 **圆角卡片 + 阴影 UI**：现代化 UI 风格，视觉观感更舒适  
- 🧱 **阿里云全家桶**：OSS 静态托管 + FC API + RDS/MySQL 排行榜

---

## 🏗️ 技术栈

### 前端

- 原生 HTML / CSS / JavaScript
- Canvas 2D 绘制游戏画面
- 响应式布局适配 PC 和移动端

### 后端 & 部署

- 阿里云 OSS：静态资源托管
- 阿里云函数计算（FC）：排行榜 API（Node.js 16）
- 阿里云 RDS / MySQL：`leaderboard` 排行表
- GitHub Actions：自动部署到 OSS
